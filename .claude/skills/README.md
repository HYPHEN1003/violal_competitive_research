# Skills 一覧

## コアスキル（全プロジェクト共通）

実装フェーズで必ず使うスキル。

| スキル | 用途 |
|--------|------|
| **design-to-code** | テキスト記述やスクリーンショットからコード変換 |
| **figma-to-code** | Figma URL → コード変換（Figma MCP経由） |
| **create-lp-section** | LPセクション（Hero/About/FAQ等）の実装 |
| **design-system-architect** | デザイントークン・コンポーネント体系の構築 |
| **accessibility-auditor** | WCAG 2.2 AA準拠チェック |

## design-to-code vs figma-to-code

| | design-to-code | figma-to-code |
|---|---|---|
| **入力** | テキスト記述、スクリーンショット、手書きスケッチ | Figma URL（ファイル/フレーム） |
| **Figma MCP** | 不要 | 必須 |
| **使い所** | Figma不使用の案件、ラフなイメージからの実装 | Figmaでデザインカンプがある案件 |
| **精度** | 記述の詳細度に依存 | デザインデータから高精度に変換 |
